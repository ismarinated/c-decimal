# c-decimal
Групповой проект по языку C. Реализация типа данных decimal и библиотеки для работы с ним. Мною реализованы функции преобразования from_int_to_decimal(), from_float_to_decimal(), from_decimal_to_int() и from_decimal_to_float(). Функции покрыты тестами, запуск выполняется через Makefile.
